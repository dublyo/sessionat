About this server

The MCP server runs inside Sessionat itself — no external Node process, no Mojo IPC. Tools call directly into Sessionat's WorkspaceService and VisitAnalyticsService.

Bound to 127.0.0.1 only. Bearer-token authenticated. Write tools (create workspace, switch, open URL) require per-client first-use approval.
