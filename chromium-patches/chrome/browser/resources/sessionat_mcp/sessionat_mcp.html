About this server

The MCP server runs inside Sessionat itself — no external Node process, no Mojo IPC. Tools call directly into Sessionat's WorkspaceService and VisitAnalyticsService.

Bound to 127.0.0.1 only. Bearer-token authenticated. v0.2 is read-only; write tools (create workspace, switch, open URL) ship in v0.3 with per-client first-use approval.
